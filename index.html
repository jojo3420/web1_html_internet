The World Wide Web (abbreviated WWW or the Web) is an information space where documents and other web resource are idemtofoed by Uniform Resource Locators(URLs), interlinked by hypertext links, and can be accesses via the Internet. English scientist Tim Berners-Lee invented the World Wide Web in 1989.
